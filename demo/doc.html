UISearchForm

  • 使用

    <ui-search-form></ui-search-form>

  • 属性

    属性名 默认值 是否必须 描述
    ref $searchForm 否 类似id, 用于在scope中获取

  • 方法

    方法名 描述
    search(other[额外的查询数据, json格式]) 调用$table.search方法, 进行查询数据
    reset() 情况所有查询控件的数据
    formData() 返回[{name: .., value:..}...]类型的数据
    formParamData() 返回name=value&name=value...类型的数据
    submit(callback[回调函数]) 监听提交事件

  • 事件

    事件名 参数 触发
    uisearchform.doSubmit (evt, formData[表单数据]) 当点击查询按钮触发
