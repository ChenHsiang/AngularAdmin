UISearchForm

  • 使用

    <ui-search-form></ui-search-form>

  • 属性

    属性名 默认值 是否必须 描述
    ref $searchForm 否 类似id, 用于在scope中获取

  • 方法

    方法名 描述
    search(other[额外的查询数据, json格式]) 调用$table.search方法, 进行查询数据
    reset() 情况所有查询控件的数据
    formData() 返回[{name: .., value:..}...]类型的数据
    formParamData() 返回name=value&name=value...类型的数据
    submit(callback[回调函数]) 监听提交事件

  • 事件

    事件名 参数 触发
    uisearchform.doSubmit (evt, formData[表单数据]) 当点击查询按钮触发

    • UITable

  • 使用

    <ui-table></ui-table>

  • 属性

    属性名 默认值 是否必须 描述
    ref $table 否 类似id, 用于在scope中获取
    state-key 否 如果设置这个, 将会把表格的排序, 列选择给缓存, 保证下次访问
    nopage false 否 是否分页
    manual false 否 false为手动请求数据, true为自动请求数据

  • 方法

    方法名 描述
    getDataAtIndex(rowIndex) 获取某行的数据
    getIdByIndex(rowIndex) 获取某行主键值(只有添加了选择列才会获取的到)
    getIdByData(rowData) 获取数据的主键值(只有添加了选择列才会获取的到)
    containItem(rowData) 当然数据集中是否包含该数据
    getDomsByColumnIndex(rowIndex) 获取某一列的所有td
    refresh(params[查询参数], url[换一个请求地址]) 只会刷新当前页
    jumpTo(index[页数]) 跳转指定页面
    getCurrentPage() 获取当前页
    search(params[查询参数], url[换一个请求地址]) 刷新跳转到第一页
    cleanSelect(isAll[是否情况全选包括跨页的])
    selectItems 获取选中的数据集合
    selectValues 获取选中的数据ID集合

  • 事件

    事件名 参数 触发
    uitable.dataSuccess (evt) 当数据加载成功触发
    uitable.renderSuccess (evt) 当数据渲染完成触发
    uitable.dateFail (evt) 当数据加载失败触发
    uitable.refreshData (evt) 当刷新数据之前时候触发

    • UITree

  • 使用

    <ui-tree>
    </ui-tree>

  • 属性

    属性名 默认值 是否必须 描述
    url 是 获取树的数据
    on-before-click 否 判断是否可以点击, 回调函数名称, 要挂载在顶层controller里面
    on-click 否 点击时触发, 回调函数名称, 要挂载在顶层controller里面
    on-before-check 否 判断是否可以选中, 回调函数名称, 要挂载在顶层controller里面
    on-check 否 选中时触发, 回调函数名称, 要挂载在顶层controller里面
    add false 否 当移动到树节点的时候会显示一个添加按钮
    edit false 否 当移动到树节点的时候会显示一个编辑按钮
    del false 否 当移动到树节点的时候会显示一个删除按钮

  • 方法

    方法名 描述
    addTreeNodeBtn(label[标题], className[class名称], handler[点击后回调]) 添加一个当移动到数节点上面的额外按钮
    addTreeNodeBtnAtIndex(index[位置],label[标题], className[class名称], handler[点击后回调]) 添加一个当移动到数节点上面的额外按钮到指定的位置
    setData(data) 手动设置树的数据
    expand(level[级别]) 按级别展开树
    expandById(id) 按id展开树
    selectItems 获取树选中的数据集合
    selectValues 获取树选中的数据ID集合
