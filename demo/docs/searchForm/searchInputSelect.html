UISearchInputSelect

  • 使用

    <ui-search-input-select></ui-search-input-select>

  • 属性

    属性名 默认值 是否必须 描述
    ref $searchInputSelect 否 类似id, 用于在scope中获取
    select-name 否 和inputName必须同时设置, 获取同时不设置(那么input的name将会设置为select的value)
    input-name 否 和inputName必须同时设置, 获取同时不设置(那么input的name将会设置为select的value)
    mode datetime 否 date为日期,time为事件,datetime为日期+时间
